# D2R Traderie Tracker

디아블로 2 레저렉션(D2R)에서 아이템 툴팁을 캡처해 [Traderie](https://traderie.com) 시세를 즉시 조회해주는 Windows 앱입니다.

---

## 설명 영상

[![D2R Traderie Tracker 소개](https://img.youtube.com/vi/p_hbEzvmv1A/maxresdefault.jpg)](https://youtu.be/p_hbEzvmv1A)

---

## 주요 기능

- 게임 내 아이템에 마우스를 올리고 핫키를 누르면 툴팁을 자동 인식해 시세를 오버레이로 표시
- PaddleOCR 기반 로컬 인식 — 인터넷 없이 작동, 캡처 이미지를 외부 서버로 전송하지 않음
- 즐겨찾기 등록 후 주기적 자동 갱신 및 오버레이 순환 표시
- 래더/논래더, 소프트코어/하드코어, 시즌별 시세 구분 지원

---

## 안전성 안내

이 프로그램에 대해 걱정하실 수 있는 부분을 미리 안내드립니다.

**개인정보 및 데이터**
- 캡처한 화면 이미지는 내 PC에서만 처리되며 외부로 전송되지 않습니다.
- API 키(선택 입력)는 로컬 파일에만 저장됩니다.
- 수집하는 개인정보가 없습니다.

**게임 보안**
- 게임 메모리를 읽거나 조작하지 않습니다.
- 화면 캡처와 Traderie 웹사이트 조회만 수행합니다.
- D2R 게임 클라이언트와 직접적인 상호작용이 없습니다.

**Windows Defender / 백신 경고**
- 소규모 배포 앱 특성상 코드 서명 인증서가 없어 경고가 표시될 수 있습니다.
- 소스 코드 전체가 이 저장소에 공개되어 있으니 직접 확인하실 수 있습니다.
