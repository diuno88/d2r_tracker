# D2R Traderie Tracker

디아블로2 레저렉션 아이템 위에 마우스를 올리고 지정한 키를 누르면  
자동으로 툴팁 감지 → OCR 분석 → Traderie 가격 조회 → 오버레이 알림으로 결과를 보여주는 프로그램

자세한 문서는 [docs/](docs/README.md) 폴더를 참고하세요.

- [docs/user-guide.md](docs/user-guide.md) — 설치, 사용법, 설정, 문제 해결
- [docs/development.md](docs/development.md) — 개발 환경 설정, 빌드, 배포
- [docs/ocr-setup.md](docs/ocr-setup.md) — OCR 방식 선택 및 AI API 키 설정

**빠른 시작 (개발 환경):**
```
scripts\install.bat   ← 최초 1회 설치
scripts\run.bat       ← 실행
```
